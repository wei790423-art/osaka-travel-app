# Osaka 8 Days Travel App

一個靜態單頁旅遊 App，規劃日本大阪 8 天 7 夜行程。

## 功能

- 每日行程與交通動線
- 依區域篩選大阪、京都、奈良、神戶、環球影城
- 旅行步調調整
- 日圓與台幣預算換算
- 票券建議與打包清單

## 本機預覽

```powershell
node server.cjs
```

打開：

```text
http://127.0.0.1:4177
```
