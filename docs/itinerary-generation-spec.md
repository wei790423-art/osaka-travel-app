# 旅行行程生成規格書

本規格提供給「旅行行程生成 Agent」使用。請嚴格依照此格式輸出，讓旅行 App 可以正確匯入每日行程、住宿、交通、餐食、照片與導航景點。

## 核心原則

1. 請輸出純 Markdown 文字，不要輸出 JSON、表格或程式碼區塊。
2. 每一天都必須獨立寫成 `## Day N 城市｜主題`。
3. 不要使用 `Day 1～Day 5 上海` 這種區段標題當作行程標題。
4. 不要把早餐、午餐、晚餐混在景點清單裡。
5. 導航景點必須放在每天的 `📍導航景點` 區塊。
6. 預算、注意事項、穿搭、美食總整理只能放在每日行程後方，不要插在每日中間。
7. 不要輸出「以下已依照規格書整理」這類說明句，第一行請直接寫旅行名稱。
8. 每日預算只能寫單一數字，不要寫 `NT$8,000～15,000` 這種區間；總花費估算請放在每日行程後方。

## 建議輸出順序

```markdown
# 上海 5 天 + 雲南 10 天（共 15 天）

國家：中國
主要城市：上海＋雲南
天數：15天
旅行類型：第一次自由行

## Day 1 上海｜抵達上海

照片：https://example.com/photo.jpg
住宿：上海外灘附近飯店
移動方式：飛機、地鐵、步行
移動路線：機場 → 飯店 → 外灘 → 南京東路

早餐：
午餐：
晚餐：生煎包、小籠包

行程：
- 抵達上海浦東機場
- 前往飯店寄放行李
- 外灘夜景
- 南京東路步行街

📍導航景點
- 外灘
- 南京東路步行街
- 和平飯店

備註：第一天不要排太滿，保留交通與入住時間。
預算：3000
```

## 每日欄位規則

### Day 標題

正確：

```markdown
## Day 1 上海｜抵達上海
## Day 2 上海｜經典上海
## Day 6 昆明｜石林與市區
## Day 15 香格里拉｜返程
```

錯誤：

```markdown
### Day 1～Day 5 上海（5天）
# Day 6～Day 15 雲南（10天）
```

如果要說明大區域，可以寫在總覽，不要用 Day 標題格式。

### 住宿

每天都要寫住宿，即使同一家飯店也要每天重複寫。

```markdown
住宿：上海外灘附近飯店
住宿：大理古城客棧
住宿：麗江古城周邊飯店
```

### 移動

請拆成移動方式與移動路線。

```markdown
移動方式：高鐵、步行
移動路線：昆明站 → 大理站 → 大理古城
```

### 餐食

早餐、午餐、晚餐必須獨立欄位。不要寫進 `📍導航景點`。

```markdown
早餐：飯店早餐
午餐：豫園附近小籠包
晚餐：雲南野生菌火鍋
```

如果不確定餐廳名稱，可以留空：

```markdown
早餐：
午餐：
晚餐：
```

### 行程

行程是當天實際時間軸，可以包含交通、活動、休息，但不要放總整理文章。
`行程：` 可以作為區塊標籤，但標籤下方必須使用項目清單，不能把 `行程：` 當成一個項目。

```markdown
行程：
- 09:00 飯店出發
- 10:00 豫園
- 13:30 城隍廟商圈
- 16:00 新天地
- 19:00 晚餐
```

### 導航景點

導航景點只放「需要開 Google 地圖導航的確定地點」。餐廳除非是確定店名，否則不要放。

```markdown
📍導航景點
- 豫園
- 城隍廟商圈
- 新天地
- 田子坊
```

錯誤：

```markdown
📍導航景點
- 早餐
- 午餐
- 晚餐
- 飯店早餐
```

### 照片

如果要放照片，每天最多放 1 張代表圖。請用這種格式：

```markdown
照片：https://example.com/day1.jpg
```

不要在同一天塞很多張 `![Image](...)`，容易讓行程文字變很長。

### 預算

每日預算只寫數字，不要寫幣別符號。

```markdown
預算：3500
```

錯誤：

```markdown
預算：NT$8,000～15,000
```

## 多城市行程規則

多城市行程請每一天都明確寫城市，不要只寫區段。

正確：

```markdown
## Day 1 上海｜抵達上海
## Day 2 上海｜經典市區
## Day 3 上海｜陸家嘴
## Day 4 上海｜迪士尼
## Day 5 上海｜前往昆明
## Day 6 昆明｜石林與翠湖
## Day 7 大理｜大理古城
## Day 8 大理｜洱海環湖
## Day 9 大理｜蒼山與古城
## Day 10 麗江｜麗江古城
## Day 11 麗江｜玉龍雪山
## Day 12 麗江｜古鎮深度遊
## Day 13 香格里拉｜獨克宗古城
## Day 14 香格里拉｜松贊林寺與納帕海
## Day 15 香格里拉｜返程
```

錯誤：

```markdown
### Day 1～Day 5 上海
### Day 6～Day 15 雲南
```

## 禁止輸出內容

請不要輸出以下內容，或放在每日行程中間：

- 大量圖片
- Markdown 表格
- 參考資料腳註
- 廣告連結
- 很長的攻略文章
- `Day 1～Day 5` 這種區段標題
- 把餐食列入導航景點
- 把住宿飯店寫成一般景點，除非當天要特別導航到飯店

住宿：上海外灘附近飯店
移動方式：飛機、地鐵、步行
移動路線：上海浦東機場 → 飯店 → 外灘 → 南京東路

早餐：
午餐：
晚餐：生煎包、小籠包

行程：
- 抵達上海

住宿：上海外灘附近飯店
移動方式：地鐵、步行

- 豫園
- 城隍廟商圈

## 給生成 Agent 的最後檢查

輸出前請確認：

- 每一天都有 `## Day N 城市｜主題`
- Day 數字連續不中斷
- 每一天都有住宿欄位
- 每一天都有早餐、午餐、晚餐欄位
- 每一天都有 `行程：`
- 每一天都有 `📍導航景點`
- 導航景點只包含確定地點
- 不使用 Day 範圍當標題
- 不使用表格放每日行程
- 不把總整理段落插入每日行程中間
